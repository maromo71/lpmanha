# Programação em C - Curso Técnico, Tecnológico e Superior

Bem-vindo ao repositório de materiais e códigos do curso de Linguagem de Programação em C, ministrado pelo Prof. Marcos Roberto de Moraes, também conhecido como Maromo. Este repositório foi criado com o objetivo de fornecer recursos educacionais para estudantes e entusiastas que desejam solucionar problemas utilizando a lógica de programação por meio da linguagem C.

## Sobre o Curso

Este curso abrange desde conceitos fundamentais até técnicas avançadas de programação em C, preparando os estudantes para desenvolver programas eficientes e robustos. Os tópicos incluem:

- **Variáveis e Constantes**: Entendendo tipos de dados e como utilizá-los.
- **Operadores e Expressões**: Aprendendo a realizar operações e construir expressões lógicas.
- **Comando de Desvio**: Instruções `if`, `else`, e `switch` para controle de fluxo.
- **Controle de Malhas**: Utilizando `for`, `while`, e `do-while` para repetições.
- **Vetores e Ponteiros**: Manipulação de arrays e endereçamentos de memória.
- **Funções de Biblioteca**: Explorando as bibliotecas padrão do C.
- **Estruturas, Uniões e Tipos Definidos pelo Usuário**: Organizando dados complexos.
- **Manipulação de Arquivos**: Leitura e escrita de arquivos para persistência de dados.

## Objetivo

Capacitar os alunos a solucionar problemas práticos por meio da implementação de programas em linguagem C, aplicando conceitos de lógica de programação, estruturas de dados e técnicas de desenvolvimento de software.

## Como Usar Este Repositório

O repositório está organizado por tópicos, seguindo a estrutura do curso. Cada diretório contém materiais didáticos, exemplos de códigos e exercícios práticos. Recomenda-se seguir a ordem dos diretórios para uma aprendizagem progressiva.

- **Diretórios**: Navegue pelos diretórios correspondentes a cada tópico para acessar os materiais. Identificado por aulaDDMM.
- **Exemplos de Código**: Estude e execute os exemplos para entender como aplicar os conceitos.
- **Exercícios**: Pratique resolvendo os exercícios propostos para consolidar seu aprendizado.
